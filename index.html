Instafetch.js

A 3KB JavaScript plugin for the Instagram API (without jQuery).

Download on GitHub

Why this plugin?

No jQuery. Why use an 87KB file only to write more JavaScript that is only being used for AJAX to return some JSONP? Let's use fetch instead. Instafetch.js is a 3KB file that returns your Instagram feed with the option of returning captions.
